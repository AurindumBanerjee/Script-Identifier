Script-Identifier

Scene Text Script Classification using ML & Deep Learning

🧠 About the Project

Script-Identifier is an ML pipeline to classify the script of scene text images. It supports traditional and deep-learning feature extractors.

  • 📌 Binary & Multiclass Classification
  • 📐 Backbones: HOG, SIFT, ResNet, VGG, ViT
  • 🧪 Dimensionality Reduction: PCA, LDA
  • 🗃 Dataset: Bharat Scene Text Dataset

⚙️ How It Works

Features are extracted from images using the selected backbone. These are stored and used to train a Logistic Regression classifier.

# Example config (logreg.yaml)
logreg_params:
  use_pca: true
  use_lda: false
  exp_name: logreg_hog

dataset:
  root_dir: data/recognition
  backbone: hog

target:
  language: hindi

📁 Project Structure

  • dataset/ – Dataset loaders & feature extractors
  • models/ – ANN & Logistic Regression logic
  • tools/ – Train/test pipeline for all models
  • config/ – YAML configs for experiments

🚀 How to Run

# Train with HOG features
python -m models.Logistic.LogBackBone

# ANN training on ViT features
python -m tools.ann_train
