Script-Identifier

Scene Text Script Classification using ML & Deep Learning Backbones

🧠 About the Project

Script-Identifier is a machine learning pipeline designed to classify the script of cropped text images from real-world scene text datasets. We support traditional and deep learning features with a flexible backbone architecture.

  • 🔍 Logistic Regression classifier
  • 📐 Feature extraction using HOG, SIFT, VGG, ResNet, ViT
  • 🧪 Dimensionality Reduction via PCA and LDA
  • 📊 Supports Binary and Multiclass classification
  • 🗂 Dataset: Bharat Scene Text Dataset

⚙️ How It Works

Features are extracted using the selected backbone and stored as latent representations. These are then used to train a logistic regression classifier.

# Example YAML config (logreg.yaml) logreg_params: use_pca: true use_lda: false exp_name: logreg_hog lda_mode: binary dataset: root_dir: data/recognition backbone: hog transformation: true target: language: hindi

📁 Project Structure

The project follows a modular structure:

  • dataset/ – Custom PyTorch datasets with feature backbones
  • models/ – Logistic regression, ANN, backbone modules
  • tools/ – Training scripts for various models
  • config/ – YAML config files
  • plots/ – Output visualizations

🚀 Running the Project

Make sure you're in the project root directory and run:

# For Logistic Regression with backbone python -m models.Logistic.LogBackBone # For ANN training python -m tools.ann_train
