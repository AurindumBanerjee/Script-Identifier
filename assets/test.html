Script-Identifier

Scene Text Script Classification using Traditional Machine Learning Techniques

Script-Identifier is a Pattern Recognition and Machine Learning (CSL2050) course project developed under the CSL2050 curriculum at IIT Jodhpur. The objective is to identify the script of text found in scene images using traditional machine learning algorithms and feature extraction techniques. Emphasis is placed on performance comparison, evaluation rigor, and model interpretability.

  • Classification across 13 Indian scripts
  • Feature extractors: HOG, SIFT, ResNet, VGG, Vision Transformer (ViT)
  • ML Models: Logistic Regression, SVM, KNN, ANN, Naive Bayes, Decision Trees, XGBoost
  • Dimensionality reduction: PCA, LDA
  • Visualizations: t-SNE, PCA plots, class distributions, decision boundaries
  • Modular code with YAML-based configuration
  • Dataset: Bharat Scene Text Dataset

Each image is passed through a configurable feature extractor (e.g., HOG or ViT) to compute latent representations. These are then fed into classical ML classifiers. Dimensionality reduction and visualization techniques help understand class separability and model behavior.

  • config/ – YAML experiment configurations
  • dataset/ – Custom PyTorch dataset and preprocessing
  • models/ – Traditional ML models and feature backbones
  • tools/ – Training and evaluation scripts
  • utils/ – Utility functions and plotting
  • Visualisation/ – t-SNE, PCA, and boundary visualizations
  • frontend/ – Next.js-based UI
  • main.py, ui.py, fastapi_server.py
