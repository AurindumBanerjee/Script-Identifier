Script-Identifier

Script Classification for Scene Text using Traditional ML Techniques

About the Project

Script-Identifier is a Pattern Recognition and Machine Learning (CSL2050) course project focused on identifying the script of text found in natural scene images. The project utilizes traditional ML techniques and compares multiple feature extraction approaches.

  • Classification across 14 Indian scripts
  • Feature extractors: HOG, SIFT, ResNet, VGG, ViT
  • ML Models: Logistic Regression, SVM, KNN, ANN, Naive Bayes, Decision Trees, XGBoost
  • Dimensionality reduction: PCA, LDA
  • Dataset: Bharat Scene Text Dataset

How It Works

Images are passed through a configurable backbone (e.g., HOG or ViT) to extract features. These are used to train traditional ML classifiers. Dimensionality reduction and visualizations support interpretation and evaluation.

# Sample logreg.yaml
logreg_params:
  use_pca: true
  use_lda: false
  exp_name: logreg_hog

dataset:
  root_dir: data/recognition
  backbone: hog

target:
  language: hindi

Project Structure

  • dataset/ – Custom PyTorch dataset and image preprocessing
  • models/ – Traditional ML and deep feature backbones
  • tools/ – Training and inference pipelines
  • config/ – YAML experiment configurations
  • frontend/ – Next.js-based demo UI

How to Run

# Run Logistic Regression
python -m models.logistic.LogBackBone

# Train ANN on ViT features
python -m tools.ann_train

# Launch API server
python fastapi_server.py

# Gradio demo UI
python ui.py
